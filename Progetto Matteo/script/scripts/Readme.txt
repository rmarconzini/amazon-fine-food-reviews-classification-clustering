GETTING STARTED:

-Avviare '1.0_Text_preprocessing.ipynb' tramite Google Colaboratory.

PREREQUISITIES:


-Installare rispettivamente le seguenti versioni delle librerie PYTHON:

Numpy==1.19.5
matplotlib==3.2.2
sklearn==1.0.2
Yellowbrick == 1.3.post1
Seaborn == 0.11.2

-Caricare in una cartella drive selezionata il file 'aclImdb_v1.tar' presente al link: [http://ai.stanford.edu/~amaas/data/sentiment/].

- Settare il path della cartella scelta all'interno del comando COPYFILE: 'copyfile('/content/gdrive/...../aclImdb_v1.tar', 'aclImdb_v1.tar')'

- Settare il path della cartella scelta all'interno del comando pd.read_csv per poter leggere i differenti dataset: ['/content/gdrive/...../df..']
